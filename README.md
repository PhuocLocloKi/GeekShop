# ✨ What Makes This Project Special?

Instead of relying on heavy 3D engines, this project creates a smooth cinematic dragon using only **HTML**, **CSS**, **SVG**, and **Vanilla JavaScript**.

---

## 🐉 Kinetic Segmented Body *(Math Physics)*

- 40 independent SVG body segments.
- Uses **Math.atan2()** and distance calculations.
- Creates realistic snake-like movement.

---

## 🔥 SVG Filters & Animated Gradients

- Advanced **SVG Filters**.
- Animated magma gradients.
- Pulsating glowing eyes.
- Dynamic fire aura.

---

## ✨ Canvas Fire Particle System

- Real-time sparks.
- Random velocities.
- Particle lifetime & fading.
- Heat trail effects.

---

## 🪽 Wing Flapping Simulation

- Wings move using **Math.sin()**.
- Natural flying animation.
- Smooth scaling and rotation.

---

## 🌙 Auto Idle Mode

After **2 seconds** without user interaction, the dragon automatically flies around the screen with a smooth idle animation.

---

# 🚀 Technologies

- HTML5
- CSS3
- SVG
- HTML5 Canvas
- Vanilla JavaScript
